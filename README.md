# Embedded Projects

This repository records my STM32 embedded learning projects.

## Hardware

- Board: STM32 Nucleo-F103RB
- MCU: STM32F103RBT6
- Development environment: macOS + STM32CubeMX + STM32CubeIDE

## Projects

### 1. led_blink

GPIO output demo.

Function:

- Configure PA5 as GPIO output
- Toggle LD2 LED periodically

### 2. uart_test

USART2 serial communication demo.

Function:

- Configure USART2 with 115200 baud rate
- Send message to Mac serial terminal
- Toggle LD2 every 1 second

## Learning Notes

This repository is used to record:

- STM32 project creation process
- GPIO configuration
- UART communication
